Information

MoneyNest4You is a free and open source application. All rights are reserved, check license for detail.

Tired of keep tracking of your daily expenditures? Well, you don't have to worry about that anymore. MoneyNest4You is here to help. With a user-friendly interface and simple functionalities, your daily tracking can be done easily. Expenditures can be enter, and save. MoneyNest4you will calculate all your expenditures, give you your total. It also predict your next expense amount according to the expenses you have entered. If no empty row is available, insert new row to enter new data. Data are saved in a text file, and can be use for track later. This feature has not been implemented.

This is application has only been tested on Linux platform, and was not fully tested. There are bugs that have not been fixed or reported, be advised.

Requirements

To use MoneyNest4You, you must download and install Eclipse or NetBeans IDE. Click on the links below to download.

  • Eclipse
  • NetBeans IDE

Once Eclipse or NetBeans IDE is installed, download MoneyNest4You files from here.

Run MoneyNest4You

The following instruction is for Eclipse user. To run on NetBeans IDE, it is similar.

To run MoneyNest4You.

  • Open Eclipse
  • Create new project, name it MoneyNest4You
  • Place all files downloaded from earlier and place them in the "src" folder.
  • Select Run
  • Select Run As
  • Select Java Application

You are now ready to use MoneyNest4You. If you have troubleshoot, email us MoneyNest4You@gmail.com

MoneyNest4You Demonstration

After successfully running MoneyNest4You, you will see the following window.

Enter data into the table.

Insert new row, click "Insert Row".

Calculate total, click "Calculate".
Your total should be seen in the Total row.

To see your predicted transaction, click on "Predict".
You will see a pop-up window with you predicted amount.

To save your data, select File, Save. A MoneyNest4You.txt file will be created with all your entered data in it.
